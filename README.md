# Energy Grid Optimizer

A Python library for solving energy grid optimization problems using linear programming.

## 🚧 Project Status

Currently under active development as part of learning modern Python software engineering practices.

## 🎯 Goals

- Build production-quality optimization software
- Implement TDD and CI/CD workflows
- Apply clean code and SOLID principles
- Create reusable, well-tested components

## 🛠️ Planned Features

- [ ] Linear programming solver integration
- [ ] Energy grid modeling components
- [ ] Multi-period optimization
- [ ] Comprehensive test suite (target: 80%+ coverage)
- [ ] CLI interface
- [ ] API documentation

## 📦 Tech Stack

- Python 3.9+
- pytest for testing
- PuLP/SciPy for optimization
- GitHub Actions for CI/CD
- Black, Flake8, mypy for code quality

## 🚀 Development Setup
```bash
# Clone the repository
git clone https://github.com/yourusername/energy-optimizer.git
cd energy-optimizer

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run tests
pytest
```
